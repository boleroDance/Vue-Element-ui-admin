VUE+ element UI + admin中后台管理系统搭建

## vueAdmin-template

###### 主要技术栈

+ vue-cli webpack
+ element-ui vue2.0 pc UI框架
+ axios 负责数据请求
+ js-cookie  轻量的JavaScript库来处理cookie
+ echarts 图标库处理数据可视化
+ mock.js 模拟数据可视化的数据
